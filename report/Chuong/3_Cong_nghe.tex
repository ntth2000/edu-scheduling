Dựa trên các yêu cầu xác định ở Chương 2, hệ thống EduSchedule được xây dựng theo kiến trúc client–server tách biệt với các công nghệ sau.

\section{Công nghệ phát triển}
\label{section:3.1}

\subsection{Backend: Spring Boot}
\label{section:3.1.1}

Spring Boot là framework Java xây dựng trên nền Spring Framework, cung cấp cơ chế tự động cấu hình giúp khởi tạo ứng dụng RESTful API nhanh chóng. Hệ thống sử dụng Java 17 kết hợp Spring Boot 4 để xây dựng toàn bộ tầng backend, bao gồm xử lý nghiệp vụ (kiểm tra xung đột, phân công giảng dạy), xác thực với Spring Security và JWT (thư viện JJWT), truy cập dữ liệu qua Spring Data JPA, và quản lý schema bằng Flyway.

\subsection{Frontend: Next.js}
\label{section:3.1.2}

Next.js là framework React hỗ trợ Server-Side Rendering, định tuyến theo App Router và Route Handler — cho phép viết API endpoint ngay trong dự án frontend. Hệ thống sử dụng Next.js 16 với TypeScript, trong đó Route Handler xử lý đăng nhập và lưu JWT vào httpOnly cookie; middleware (proxy) kiểm soát phân quyền route; trang tra cứu thời khoá biểu công khai được render phía server.

\subsection{Cơ sở dữ liệu: PostgreSQL}
\label{section:3.1.3}

PostgreSQL là hệ quản trị cơ sở dữ liệu quan hệ mã nguồn mở, hỗ trợ đầy đủ ràng buộc toàn vẹn tham chiếu và transaction ACID. Dữ liệu của hệ thống gồm các entity có quan hệ chặt chẽ (Teacher, SchoolClass, Subject, Assignment, Timetable), đòi hỏi mô hình quan hệ và ràng buộc khoá ngoại để đảm bảo nhất quán khi thêm, sửa, xoá.


\section{Công cụ triển khai và kiểm thử}
\label{section:3.2}

Hệ thống được triển khai và kiểm thử thông qua các công cụ miễn phí sau:
\begin{itemize}[noitemsep]
    \item \textbf{Vercel}: nền tảng hosting cho frontend Next.js, cung cấp HTTPS và CI/CD tự động từ Git repository.
    \item \textbf{Docker}: đóng gói backend Spring Boot thành container, đảm bảo môi trường chạy nhất quán giữa development và production, triển khai trên nền tảng miễn phí.
    \item \textbf{Postman}: công cụ kiểm thử API, dùng để kiểm tra và xác minh các endpoint backend trong quá trình phát triển.
\end{itemize}

Chương này đã trình bày các công nghệ xây dựng hệ thống EduSchedule, bao gồm Spring Boot cho backend, Next.js cho frontend, PostgreSQL cho cơ sở dữ liệu, cùng Vercel, Docker và Postman cho triển khai và kiểm thử. Chương 4 tiếp theo sẽ trình bày thiết kế kiến trúc và quá trình hiện thực hoá hệ thống dựa trên nền tảng công nghệ này.
